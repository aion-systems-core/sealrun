# Examples

This directory contains runnable examples for core SealRun workflows.

- `basic/`: shell-based execute, replay, and drift walkthroughs
- `governance/`: policy validation and CI-oriented checks
- `sdk/`: Rust SDK-focused sample programs
- `ci/`: CI-centric example setup
- `advanced/`: expanded scenarios
- language demos: `go/`, `node/`, `csharp/`, `c_abi/`, `c_abi_showcase/`

For documentation-linked entry points, see `docs/examples/README.md`.
